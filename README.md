# React Rule Builder

Projeto inicial com React, TypeScript e Vite.

## Primeiros passos

```bash
pnpm install
pnpm dev
```

## Ambiente

```bash
node -v
pnpm -v
```

Esperado:

- Node 24+
- pnpm 10+

## Scripts

```bash
pnpm check
pnpm dev
pnpm build
pnpm lint
pnpm lint:fix
pnpm format
pnpm format:check
pnpm preview
```
